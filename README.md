This is an expense tracker application based on Model, View, Controller (MVC) architecture. 

Front End Build:
ReactJS components with Axios and React Routing,
CSS3 for stying of components and display data

Back End Build: 
-Spring Boot 
-Java 8
-MySQL DB.
